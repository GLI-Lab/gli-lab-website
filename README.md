# Welcome to Graph & Language Intelligence (GLI) Lab

- 주요연구분야 요약

    - <b>자연어처리</b>: 질의응답, 정보 검색/추출, 지식/LLM 증류 및 응용
        
        → 그래프 지식을 통합하여 LLM의 환각 문제를 줄이고, 더 일관된 응답 생성 가능

    - <b>추천시스템</b>: 지식기반/설명가능한 추천, 멀티모달/대화형 추천, 그래프 기반 추천
        
        → 그래프 기반의 사용자–아이템 관계 확장을 통해 의도와 맥락 중심의 추천 가능

    - <b>그래프 표현학습</b>: 그래프 뉴럴네트워크(GNNs), 지식그래프 표현/완성/검증/구축

        → 그래프 구축/확장 및 다양한 딥러닝 모델과의 통합을 위한 기반 기술로 활용 가능

    - <b>LLM·그래프 융합</b>: 지식기반/멀티모달 LLM, 텍스트-그래프 변환, LLM 에이전트, GraphRAG

        → 그래프를 통해 LLM에 외부 지식을 입력하여 신뢰성 높고 도메인 특화 응답 생성 가능

    - <b>이상탐지</b>: 거래 이상탐지, 사용자 행동 기반 이상탐지, 멀티모달 이상탐지

        → 그래프를 통해 행위 간의 상관관계를 모델링하여 상황 맥락에 맞는 이상 탐지 가능


- 주요연구분야 요약(영문)

    - <b>Natural Language Processing</b>: Question Answering, Information Retrieval & Extraction, Knowledge / LLM Distillation and Applications
    
        → By incorporating graph knowledge, we can curb LLM hallucinations and generate more consistent answers.

    - <b>Recommender Systems</b>: Knowledge‑enhanced & Explainable Recommendations, Multimodal / Conversational Recommendations, Graph‑based Recommendations
    
        → Expanding user‑item relations on graphs enables intent‑ and context‑centric recommendations.

    - <b>Graph Representation Learning</b>: Graph Neural Networks (GNNs), Knowledge Graph Representation / Completion / Validation / Construction
    
        → Serves as the groundwork for building / extending graphs and integrating them with diverse deep‑learning models.

    - <b>Synergizing LLMs and Graphs</b>: Knowledge‑grounded & Multimodal LLMs, Text‑to‑Graph & Graph‑to‑Text, Graph‑Driven LLM Agents, GraphRAG
    
        → Feeding external graph knowledge into LLMs yields reliable, domain‑specific responses.

    - <b>Anomaly Detection</b>: Transaction‑level Anomaly Detection, Behavior‑based User Anomaly Detection, Multimodal Anomaly Detection
        
        → Modeling correlations between actions on graphs allows context‑aware anomaly detection.
