# Welcome to Graph & Language Intelligence (GLI) Lab

- 주요연구분야 요약

    - 자연어처리: 질의응답, 정보 검색/추출, 지식/LLM 증류

    - 추천시스템: 지식기반/설명가능한 추천, 멀티모달/대화형 추천, 그래프 기반 추천

    - 그래프 표현학습: 그래프 뉴럴네트워크(GNNs), 지식그래프 표현/완성/검증/구축

    - LLM·그래프 융합: 지식기반/멀티모달 LLM, 텍스트-그래프 변환, LLM 에이전트, GraphRAG

- 주요연구분야 요약(영문)

    - Natural Language Processing: Question Answering, Information Retrieval & Extraction, Knowledge & LLM Distillation

    - Recommender Systems: Knowledge-enhanced & Explainable Recommendations, Conversational Recommendation, Graph-based Recommendation

    - Graph Representation Learning: Graph Neural Networks, Knowledge Graph Representation/Completion/Validation/Construction
    
    - Synergizing LLMs and Graphs: Knowledge-grounded & Multi-modal LLMs, Text-to-Graph & Graph-to-Text, Graph-Driven LLM Agent, GraphRAG
